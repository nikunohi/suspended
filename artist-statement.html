SUSPENDED IN A GUST,
time allowed the great machine to
reforge a part of what had crumbled away.

This tool is an interface that simulates change over time. SUSPEND helps users understand how change over time can be visualized. It allows users to examine the minute differences in the frames of a motion graphic sequence. By comparing and contrasting adjacent still frames from the sequence, users can see how differences in scale, depth, position, rotation, shape, & more can be animated.

The machine captures and examines action,
putting its operator in the boots of explorers rediscovering
moments suspended conveniently in the gusts of time.

On start, the program runs a splash-screen animation. This animation is the first part of a risograph frame-by-frame animation. The splash screen animation ends, and a button to start the program appears.

A gentle breeze can be heard blowing lightly,
bouncing off the surfaces of the great machine.

Once clicked, another transitional risograph animation plays. This eventually gives way to a secondary menu screen where users can begin their exploration. Users can use their scroll wheel to freely zoom into & more closely examine the frame. In the frame carousel, users can use the “<” & “>” keys to navigate backward one frame and forward one frame, respectively. Users can also use the “h” key to bring up a help menu, & “1” key to restart the program.
